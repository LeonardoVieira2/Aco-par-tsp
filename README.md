# Aco-par-tsp
utilizei o algoritmo Ant Colony Optimization (ACO) para encontrar uma rota  cíclica (Problema do Caixeiro-Viajante, TSP) que visita 10 pontos e retorna ao início. Os pontos  foram definidos em coordenadas de pixels da imagem do mapa e depois normalizados para  0,10,10,1 para sobreposição na figura.
